# Security Principles

Security by Default

Least Privilege

Defense in Depth

Secure by Design

Zero Trust

---

# Secrets

Never commit secrets.

Use environment variables.

Rotate credentials.

Use encrypted storage.

---

# Authentication

Separate User/Admin.

HttpOnly cookies.

Secure cookies.

Session expiration.

---

# Authorization

Role-based.

Permission checks.

No client-side authorization.

---

# Input Validation

Validate everything.

Escape output.

Sanitize input.

---

# Logging

Audit security events.

Never log secrets.
