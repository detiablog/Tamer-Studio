# Shared Principles

These principles apply to every master prompt.

## 1. Documentation First

Read documentation before implementation.

---

## 2. Reuse Before Create

Prefer extending existing modules over creating new ones.

---

## 3. Architecture First

Architecture has higher priority than implementation convenience.

---

## 4. Security by Default

Every implementation must consider security.

---

## 5. Maintainability Over Cleverness

Readable code is preferred over complex solutions.

---

## 6. AI Must Explain Decisions

When making significant implementation choices, explain:

- Why?
- Alternatives considered.
- Trade-offs.
