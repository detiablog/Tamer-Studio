# Tamer Studio AI Prompt System

Version: 2.0

## Purpose

This directory contains the official AI prompt framework used during software development.

These prompts standardize how AI assistants analyze, implement, review, and release software within the Tamer Studio ecosystem.

Every AI interaction should begin with the appropriate master prompt.
